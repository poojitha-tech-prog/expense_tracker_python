Expense Tracker in Python :-
Project Description:
This is a simple Expense Tracker developed using Python. The application allows users to add expenses, view saved expenses, and calculate total spending. Expense data is stored in a text file so that it is available even after restarting the program.
Features:
- Add new expenses
- Categorize expenses
- View all saved expenses
- Calculate total spending
- Store data using file handling
- Load previous expenses automatically
Technologies Used:
- Python
- File Handling
- Functions
- Lists
- Loops
- Conditional Statements
How to Run:
1. Open the project in Python.
2. Run "expense_tracker.py".
3. Choose options from the menu.
4. Add and manage expenses.
